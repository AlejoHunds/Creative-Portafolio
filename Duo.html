Duolingo

Interactive learning concept for contextual language practice.

Concept

This project proposes an alternative interaction mechanic for the language learning platform Duolingo. The “Fill the Gap” concept encourages contextual learning by asking users to complete sentences within realistic language scenarios instead of relying only on translation exercises.

Design Direction

The visual approach follows Duolingo’s playful identity while simplifying the interface to reduce cognitive load. Clear hierarchy, colour cues and spacing guide the user through each learning step while maintaining engagement.
